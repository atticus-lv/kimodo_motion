# Kimodo Motion — Installation

The full installation guide (Windows + macOS, text-encoder model setup,
troubleshooting, changelog) lives at the repository root:

- 中文：[INSTALL.md](../INSTALL.md)
- English: [INSTALL_EN.md](../INSTALL_EN.md)

Quick path: enable the extension, then **N-panel > Kimodo > Runtime Install >
One-click install runtime**, and set the add-on's *venv path* preference to where
the runtime was installed.
